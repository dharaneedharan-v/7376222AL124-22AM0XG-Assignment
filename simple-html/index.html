Docker Commands

Common Docker Commands
Command Description
docker run Run a new container from an image
docker build Build an image from a Dockerfile
docker ps List running containers
docker stop Stop a running container
docker rm Remove a stopped container
docker rmi Remove an image
docker logs Fetch logs from a container
docker-compose up Start services defined in a docker-compose.yml file
docker-compose down Stop and remove services, networks, and volumes

Getting Started

To get started with our Dockerized application, follow these simple steps:

  1. Clone the repository from GitHub.
  2. Build the Docker image using docker build -t my-docker-app ..
  3. Run the Docker container using docker run -d -p 8080:80 my-docker-app.
  4. Access the application at http://localhost:8080.
